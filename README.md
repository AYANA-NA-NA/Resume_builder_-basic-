# Basic Resume Builder

A simple, stylish resume builder built with HTML, CSS, and JavaScript. It lets users choose a template, fill in their details, preview the resume, and download it as a PDF.

## Features
- Clean pink-and-white UI
- Two resume templates to choose from
- Live resume preview
- PDF download support

## Project Structure
- index.html: landing page with template selection
- form.html: form for entering resume details
- style.css: styling for the app
- script.js: resume generation and PDF download logic
- templates/: resume template HTML files

## How to Run
1. Open the project folder in your browser.
2. Start with index.html.
3. Choose a template, fill in your details, and preview your resume.

## Notes
This project is a lightweight front-end app and does not require a backend server.
